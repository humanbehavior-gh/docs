### Option 1: Installation Wizard (Recommended)

The easiest way to get started is using our installation wizard:

```bash
# Navigate to your project directory
cd your-project-directory

# Run the installation wizard
npx humanbehavior-js YOUR_API_KEY
```

The wizard will:
1. Ask for your API key (if not provided as an argument)
2. Let you select **Vue** from the framework options
3. Automatically install and configure the SDK for your Vue project:
   - Install the `humanbehavior-js` package
   - Add HumanBehaviorPlugin to your main.js/ts
   - Configure environment variables
   - Set up proper Vue integration

**Version Compatibility**: The wizard works with most Vue versions and automatically detects:
- TypeScript support (`typescript` or `@vue/cli-service`)
- Router usage (`vue-router`)
- Bundlers: Vite, Webpack, esbuild, rollup
- Package managers: npm, yarn, pnpm

### Option 2: Manual Setup

If you prefer manual setup or the wizard doesn't work for your project, follow the steps below.

## Step 1: Installation

```bash
npm install humanbehavior-js
# or
yarn add humanbehavior-js
```

## Step 2: Environment Variables

Create an environment file (`.env`) with your API key:

```bash
VITE_HUMANBEHAVIOR_API_KEY=your-api-key-here
# OR for Vue CLI
VUE_APP_HUMANBEHAVIOR_API_KEY=your-api-key-here
```

## Step 3: Basic Setup

### Update main.ts

In your main `main.ts` file:

```typescript
import { createApp } from 'vue';
import { HumanBehaviorPlugin } from 'humanbehavior-js/vue';
import App from './App.vue';

const app = createApp(App);

app.use(HumanBehaviorPlugin, {
  apiKey: process.env.VITE_HUMANBEHAVIOR_API_KEY // For Vite
  // OR
  apiKey: process.env.VUE_APP_HUMANBEHAVIOR_API_KEY // For Vue CLI
});

app.mount('#app');
```

<Note>
  **That's it!** The SDK now automatically tracks user interactions, navigation, and console events. Everything past here is optional.
</Note>
