# Using HumanBehaviorTracker with Vue.js

## Quickstart

Human Behavior works seamlessly to record sessions on your Vue.js app.

## Step 1: Installation

```bash
npm install humanbehavior-js
# or
yarn add humanbehavior-js
```

## Step 2: Basic Setup

### Add the Tracker to Your App

In your main `main.js` or `main.ts` file:

```javascript
import { createApp } from 'vue'
import App from './App.vue'
import { HumanBehaviorTracker } from 'humanbehavior-js'

// Initialize the tracker once globally
const tracker = HumanBehaviorTracker.init(process.env.VUE_APP_HUMANBEHAVIOR_API_KEY)

// Create and mount the app
const app = createApp(App)
app.mount('#app')
```

<Note>
  **That's it!** The SDK now automatically tracks user interactions, navigation, and console events. Everything past here is optional.
</Note>

## Optional: Advanced Features

### Track Custom Events

```vue
<template>
  <div>
    <h1>My Component</h1>
    <button @click="handleButtonClick">Click Me</button>
  </div>
</template>

<script>
import { HumanBehaviorTracker } from 'humanbehavior-js'

export default {
  name: 'MyComponent',
  mounted() {
    // Get the global tracker instance
    const tracker = HumanBehaviorTracker.init(process.env.VUE_APP_HUMANBEHAVIOR_API_KEY)
    
    // Track component view
    tracker.customEvent('component_viewed', {
      componentName: 'MyComponent'
    })
  },
  methods: {
    async handleButtonClick() {
      const tracker = HumanBehaviorTracker.init(process.env.VUE_APP_HUMANBEHAVIOR_API_KEY)
      await tracker.customEvent('button_clicked', {
        buttonLocation: 'header'
      })
    }
  }
}
</script>
```

### User Authentication Example

```vue
<template>
  <div>
    <form @submit.prevent="handleLogin">
      <input v-model="email" type="email" placeholder="Email" />
      <input v-model="password" type="password" placeholder="Password" />
      <button type="submit">Login</button>
    </form>
  </div>
</template>

<script>
import { HumanBehaviorTracker } from 'humanbehavior-js'

export default {
  name: 'LoginForm',
  data() {
    return {
      email: '',
      password: ''
    }
  },
  methods: {
    async handleLogin() {
      try {
        // Your login logic here
        const user = await this.login(this.email, this.password)
        
        if (user) {
          // Initialize HumanBehavior tracker
          const tracker = HumanBehaviorTracker.init(process.env.VUE_APP_HUMANBEHAVIOR_API_KEY)
          
          // Identify user with current endUserId
          await tracker.identifyUser({
            userProperties: {
              email: user.email,
              name: user.name,
              userId: user.id,
              plan: user.plan
            }
          })
        }
      } catch (error) {
        console.error('Login failed:', error)
      }
    }
  }
}
</script>
```

## Troubleshooting

- Check browser console for any error messages
- Verify your API key is correct
- Ensure the tracker is initialized before any user interaction
