### Option 1: Installation Wizard (Recommended)

The easiest way to get started is using our installation wizard:

```bash
npx humanbehavior-js YOUR_API_KEY
```

The wizard will:
1. Ask for your API key (if not provided as an argument)
2. Let you select **Svelte** from the framework options
3. Automatically install and configure the SDK for your Svelte project:
   - Install the `humanbehavior-js` package
   - Add humanBehaviorStore to your main file
   - Configure environment variables
   - Set up proper Svelte integration

**Version Compatibility**: The wizard automatically detects your Svelte setup:
- **SvelteKit**: Detects `svelte.config.js`
- **Regular Svelte**: Standalone Svelte projects
- TypeScript support (`typescript` or `svelte-check`)
- Router usage (`svelte-routing` or `@sveltejs/kit`)
- Package managers: npm, yarn, pnpm

### Option 2: Manual Setup

If you prefer manual setup or the wizard doesn't work for your project, follow the steps below.

## Step 1: Installation

## Step 2: Basic Setup

### Add the Tracker to Your App

In your `main.js` or `main.ts` file:

```javascript
import App from './App.svelte';
import { humanBehaviorStore } from 'humanbehavior-js/svelte';

// Initialize the tracker once globally
const tracker = humanBehaviorStore.init(import.meta.env.VITE_HUMANBEHAVIOR_API_KEY);

const app = new App({
  target: document.body
});

export default app;
```

<Note>
  **That's it!** The SDK now automatically tracks user interactions, navigation, and console events. Everything past here is optional.
</Note>

## Optional: Advanced Features

### Track Custom Events

```svelte
<script>
  import { onMount } from 'svelte';
  import { humanBehaviorStore } from 'humanbehavior-js/svelte';

  let tracker;

  onMount(() => {
    tracker = humanBehaviorStore.init(import.meta.env.VITE_HUMANBEHAVIOR_API_KEY);
    
    // Track component view
    tracker.customEvent('component_viewed', {
      componentName: 'MyComponent'
    });
  });

  async function handleButtonClick() {
    await tracker.customEvent('button_clicked', {
      buttonLocation: 'header'
    });
  }
</script>

<div>
  <h1>My Component</h1>
  <button on:click={handleButtonClick}>Click Me</button>
</div>
```

### User Authentication Example

```svelte
<script>
  import { humanBehaviorStore } from 'humanbehavior-js/svelte';

  let email = '';
  let password = '';

  async function handleLogin() {
    try {
      // Your login logic here
      const user = await login(email, password);
      
      if (user) {
        // Initialize HumanBehavior tracker
        const tracker = humanBehaviorStore.init(import.meta.env.VITE_HUMANBEHAVIOR_API_KEY);
        
        // Identify user with current endUserId
        await tracker.identifyUser({
          userProperties: {
            email: user.email,
            name: user.name,
            userId: user.id,
            plan: user.plan
          }
        });
      }
    } catch (error) {
      console.error('Login failed:', error);
    }
  }
</script>

<div>
  <form on:submit|preventDefault={handleLogin}>
    <input bind:value={email} type="email" placeholder="Email" />
    <input bind:value={password} type="password" placeholder="Password" />
    <button type="submit">Login</button>
  </form>
</div>
```

## Troubleshooting

- Check browser console for any error messages
- Verify your API key is correct
- Ensure the tracker is initialized before any user interaction
