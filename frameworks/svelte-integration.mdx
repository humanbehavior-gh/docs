# Using HumanBehaviorTracker with Svelte

## Quickstart

Human Behavior works seamlessly to record sessions on your Svelte app.

## Step 1: Installation

```bash
npm install humanbehavior-js
# or
yarn add humanbehavior-js
```

## Step 2: Basic Setup

### Add the Tracker to Your App

In your `main.js` or `main.ts` file:

```javascript
import App from './App.svelte';
import { HumanBehaviorTracker } from 'humanbehavior-js';

// Initialize the tracker once globally
const tracker = HumanBehaviorTracker.init(process.env.VITE_HUMANBEHAVIOR_API_KEY);

const app = new App({
  target: document.body
});

export default app;
```

<Note>
  **That's it!** The SDK now automatically tracks user interactions, navigation, and console events. Everything past here is optional.
</Note>

## Optional: Advanced Features

### Track Custom Events

```svelte
<script>
  import { onMount } from 'svelte';
  import { HumanBehaviorTracker } from 'humanbehavior-js';

  let tracker;

  onMount(() => {
    tracker = HumanBehaviorTracker.init('your-api-key-here');
    
    // Track component view
    tracker.customEvent('component_viewed', {
      componentName: 'MyComponent'
    });
  });

  async function handleButtonClick() {
    await tracker.customEvent('button_clicked', {
      buttonLocation: 'header'
    });
  }
</script>

<div>
  <h1>My Component</h1>
  <button on:click={handleButtonClick}>Click Me</button>
</div>
```

### User Authentication Example

```svelte
<script>
  import { HumanBehaviorTracker } from 'humanbehavior-js';

  let email = '';
  let password = '';

  async function handleLogin() {
    try {
      // Your login logic here
      const user = await login(email, password);
      
      if (user) {
        // Initialize HumanBehavior tracker
        const tracker = HumanBehaviorTracker.init('your-api-key-here');
        
        // Identify user with current endUserId
        await tracker.identifyUser({
          userProperties: {
            email: user.email,
            name: user.name,
            userId: user.id,
            plan: user.plan
          }
        });
      }
    } catch (error) {
      console.error('Login failed:', error);
    }
  }
</script>

<div>
  <form on:submit|preventDefault={handleLogin}>
    <input bind:value={email} type="email" placeholder="Email" />
    <input bind:value={password} type="password" placeholder="Password" />
    <button type="submit">Login</button>
  </form>
</div>
```


```javascript
// In your code
const tracker = HumanBehaviorTracker.init(import.meta.env.VITE_HUMANBEHAVIOR_API_KEY);
```

## Troubleshooting

- Check browser console for any error messages
- Verify your API key is correct
- Ensure the tracker is initialized before any user interaction
